# TEKK Solutions – Consulting Website

## Overview

Full-stack consulting website for **TEKK Solutions** (Business Systems & Finance Consulting, Montreal QC). Built on Flask with Jinja2 templates, a Dark Tech CSS design system, and a standalone React + Vite portfolio SPA.

## Setup Instructions

### Prerequisites

- Python 3.10+
- Node.js 18+ (for React SPA only)

### Run the Flask App

```bash
# From the project root
pip install -r requirements.txt
python run.py
# visit http://localhost:5000
```

`run.py` automatically creates the SQLite database at `instance/db.sqlite3` on first run.

### Run the React SPA (development)

```bash
cd tekk-portfolio
npm install
npm run dev        # → http://localhost:5173
```

### Build the React SPA for Flask

```bash
cd tekk-portfolio
npm run build      # outputs to website/static/portfolio/
# then: python run.py
# visit http://localhost:5000/landing
```

## What Was Built

### 1. Flask Backend Restructuring

- **`run.py`** — top-level entry point; calls `create_app()`, runs `db.create_all()`, starts dev server
- **`website/__init__.py`** — app factory with Flask-Login, SQLAlchemy, Flask-WTF; added missing `@login_manager.user_loader` callback and `login_view = "auth.login"`
- **`website/views.py`** — Blueprint for all public routes including `/landing` and `/landing/<path:filename>` for serving the built React SPA
- **`website/forms.py`** — `ContactForm` with name, email, message, and optional file attachment (FileAllowed)
- **`website/client_dashboard.py`** — POST handler for `request.files.getlist("file[]")` with `secure_filename`

### 2. Jinja2 Templates

| Template | Description |
|---|---|
| `base.html` | Shared navbar with active-link state, login/logout auth links, flash messages, footer, `url_for` for all static assets |
| `index.html` | Full homepage: sticky navbar, hero, services grid, portfolio case studies, 3-col footer |
| `portfolio.html` | Static grid + dynamic JSON section populated by `script.js` |
| `contact.html` | FlaskForm with drag-and-drop file upload area |
| `login.html` / `signup.html` | Auth forms with CSRF tokens |
| `client_dashboard.html` | File upload card + account details |

## TEKK Solutions Portfolio SPA (`tekk-portfolio/`)

A standalone React + Vite single-page application for the high-end TEKK Solutions public-facing portfolio, served by Flask at `/landing`.

### Stack

- **React 18** + **Vite 5**
- **Tailwind CSS 3** — custom neon palette, glassmorphism utilities, `tracking-ultra`
- **Framer Motion 11** — entrance animations, scroll parallax, hover effects, `AnimatePresence`
- **Lucide React** — icons throughout

### Components

| Component | Description |
|---|---|
| `Navbar.jsx` | Fixed; transparent → glass blur on scroll; `AnimatePresence` mobile hamburger menu |
| `Hero.jsx` | `useScroll`/`useTransform` parallax bg; glassmorphism headline card; stats strip; scroll indicator |
| `About.jsx` | Two-column; animated CSS gradient panel (fallback for missing `about-bg.webp`); SAGE badge; capability checklist |
| `Services.jsx` | 3 glass cards (ERP / Data Automation / Infrastructure); `whileHover={{ y: -8 }}`; per-color neon glow |
| `Projects.jsx` | 3 image cards; Framer Motion zoom + description reveal + arrow pop-in on hover |
| `Contact.jsx` | Glass form with `submitted` state; Why TEKK sidebar; footer strip |

### Vite Configuration

```js
// vite.config.js
base: '/static/portfolio/',           // Flask serves at this path
build: { outDir: '../website/static/portfolio' }
```

Images use `import.meta.env.BASE_URL + 'images/filename.webp'` — resolves correctly in both Vite dev and Flask production.
